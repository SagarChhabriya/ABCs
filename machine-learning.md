| Step | Term                             | Simple Meaning                                                                  | Example                                       |
| ---- | -------------------------------- | ------------------------------------------------------------------------------- | --------------------------------------------- |
| 1    | **Data**                         | Any information that can be collected.                                          | Names, photos, temperatures, videos           |
| 2    | **Information**                  | Data that has meaning.                                                          | "Ahmed scored 90 marks."                      |
| 3    | **Dataset**                      | A collection of related data.                                                   | A spreadsheet containing student records      |
| 4    | **Record (Instance)**            | One complete row in a dataset.                                                  | One student's information                     |
| 5    | **Attribute (Column)**           | A type of information stored for every record.                                  | Age, Height, Salary                           |
| 6    | **Feature**                      | An input the model uses to make predictions.                                    | Age and Salary used to predict loan approval  |
| 7    | **Label (Target)**               | The correct answer the model should learn.                                      | Approved = Yes                                |
| 8    | **Structured Data**              | Data arranged in rows and columns.                                              | Excel sheet, SQL database                     |
| 9    | **Unstructured Data**            | Data without a fixed format.                                                    | Images, videos, emails, audio                 |
| 10   | **Semi-Structured Data**         | Data with some organization but not tables.                                     | JSON, XML, HTML                               |
| 11   | **Training Data**                | Data used to teach the model.                                                   | 8,000 customer records                        |
| 12   | **Validation Data**              | Data used during training to improve the model.                                 | 1,000 customer records                        |
| 13   | **Test Data**                    | Completely unseen data used for evaluation.                                     | Final 1,000 records                           |
| 14   | **Artificial Intelligence (AI)** | Making computers perform tasks that normally require human intelligence.        | Voice assistants, self-driving cars           |
| 15   | **Machine Learning (ML)**        | A branch of AI where computers learn from data instead of explicit programming. | Spam email detection                          |
| 16   | **Algorithm**                    | A method or procedure used to learn from data.                                  | Decision Tree, Linear Regression              |
| 17   | **Model**                        | The learned result after training an algorithm.                                 | A trained spam detector                       |
| 18   | **Training**                     | Teaching the model using training data.                                         | Showing thousands of labeled emails           |
| 19   | **Learning**                     | The model discovers patterns from the data.                                     | Recognizing spam words automatically          |
| 20   | **Inference (Prediction)**       | Using the trained model on new data.                                            | Predicting whether a new email is spam        |
| 21   | **Prediction**                   | The model's answer or output.                                                   | "Spam"                                        |
| 22   | **Supervised Learning**          | Learning from data that includes correct answers.                               | House price prediction                        |
| 23   | **Unsupervised Learning**        | Finding patterns without correct answers.                                       | Customer grouping                             |
| 24   | **Reinforcement Learning**       | Learning through rewards and penalties.                                         | Game-playing AI                               |
| 25   | **Classification**               | Predicting categories.                                                          | Spam or Not Spam                              |
| 26   | **Regression**                   | Predicting numbers.                                                             | Predicting house prices                       |
| 27   | **Clustering**                   | Grouping similar items together.                                                | Customer segmentation                         |
| 28   | **Accuracy**                     | Percentage of correct predictions.                                              | 95% correct                                   |
| 29   | **Loss**                         | How wrong the model's predictions are.                                          | Lower loss means better learning              |
| 30   | **Epoch**                        | One complete pass through the training dataset.                                 | Reading the entire textbook once              |
| 31   | **Batch**                        | A small portion of data processed at one time.                                  | Reading one chapter at a time                 |
| 32   | **Iteration**                    | One update of the model using one batch.                                        | One study session                             |
| 33   | **Overfitting**                  | The model memorizes training data instead of learning general rules.            | Memorizing exam answers only                  |
| 34   | **Underfitting**                 | The model hasn't learned enough.                                                | Studying only one page before an exam         |
| 35   | **Generalization**               | Performing well on new, unseen data.                                            | Solving new questions correctly               |
| 36   | **Hyperparameter**               | Settings chosen before training begins.                                         | Learning rate, batch size                     |
| 37   | **Bias**                         | Error caused by overly simple assumptions.                                      | Assuming all expensive houses are large       |
| 38   | **Variance**                     | Error caused by learning too many small details.                                | Memorizing every practice question            |
| 39   | **Neural Network**               | A model inspired by how the brain processes information.                        | Used for image recognition                    |
| 40   | **Deep Learning**                | Machine learning using neural networks with many layers.                        | ChatGPT, image generation                     |
| 41   | **Preprocessing**                | Cleaning and preparing data before training.                                    | Removing missing values                       |
| 42   | **Deployment**                   | Making the trained model available in a real application.                       | A recommendation system on a shopping website |
